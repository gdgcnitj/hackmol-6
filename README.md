# HackMol 6

A clean and modern website for the sixth edition of HackMol, designed to provide all important event details in one place.

## About

HackMol 6 is a hackathon that brings together developers, designers, and innovators to build impactful solutions.
This website acts as the central hub for announcements, registrations, and event updates.
